#  Primalux Smartshop

E-commerce multicategoría desarrollado como Trabajo de Fin de Grado (TFG), enfocado en crear una plataforma de compra online funcional, con autenticación segura (2FA), estructura MVC y un sistema de recomendaciones sencillo basado en la actividad del usuario.

---

##  Objetivo del proyecto

El objetivo principal es desarrollar una aplicación web que permita gestionar un proceso completo de compra online, aplicando una estructura organizada y teniendo en cuenta aspectos como la seguridad y la experiencia de usuario.

---

##  Enfoque

El proyecto se centra en construir una base sólida y funcional, priorizando:

- Organización del código mediante MVC  
- Seguridad en la autenticación de usuarios  
- Flujo completo de compra  
- Estructura preparada para futuras mejoras  

---

##  Tecnologías utilizadas

- Backend: PHP  
- Frontend: HTML, CSS y JavaScript  
- Base de datos: MySQL  
- Entorno de desarrollo: Docker  
- Despliegue: IONOS  

---

##  Seguridad

Se implementa autenticación con doble factor (2FA) mediante código enviado al correo electrónico del usuario, junto con almacenamiento seguro de contraseñas y control de acceso a funcionalidades.

---

##  Funcionalidades principales

- Registro e inicio de sesión  
- Sistema de autenticación con 2FA  
- Catálogo de productos por categorías  
- Carrito de compra  
- Proceso de pedido  
- Gestión básica de pedidos  
- Sistema de recomendaciones sencillo
