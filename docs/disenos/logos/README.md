#  Logotipo del proyecto

En esta carpeta se incluyen las distintas versiones del logotipo utilizadas en el proyecto.

---

##  Versiones

- Logo para header: más simple y limpio  
- Logo para footer: más visual y llamativo  

---

##  Proceso

El logotipo se ha creado combinando ideas propias con el uso de herramientas de inteligencia artificial.

A partir de una idea inicial, se generaron varias opciones y se seleccionaron las más adecuadas para el proyecto.

---

##  Criterios

- Claridad  
- Adaptación a la web  
- Coherencia  
- Diferenciación entre zonas  

---

##  Nota

Los logos pueden ajustarse ligeramente durante el desarrollo si se detecta alguna mejora en su adaptación dentro de la aplicación.
